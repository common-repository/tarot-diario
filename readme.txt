Reciba una carta de tarot en su panel de administrador.

== Description ==

Te encanta la astrologia tanto como a nosotros? Entonces este es tu 
plugin.
Especialmente creado para tarotistas, cada vez que te loguees como administrador 
recibiras el nombre de una carta del tarot.

Puntualmente una de estas cartas:

El Loco</br>
El Mago</br>
La Sacerdotisa</br>
La Emperatriz</br>
El Emperador</br>
El sumo sacerdote</br>
Los enamorados</br>
El carro</br>
La justicia</br>
El ermitano</br>
Rueda de la fortuna</br>
La fuerza</br>
El colgado</br>
La muerte</br>
La templanza</br>
El diablo</br>
La Torre</br>
La Estrella</br>
La Luna</br>
El Sol</br>
El Juicio</br>
El mundo</br>

La interpretacion, desde luego, debe hacerla uno mismo. Por eso es que esta 
recomendado mas que nada a quienes realizan tiradas de tarot.


== Installation ==

Siga las instrucciones para instalar su plugin "Tarot diario":

1. Suba la carpeta `Tarot diario` al directorio `/wp-content/plugins/`
2. Active el plugin desde el menu de wordpress


== Frequently Asked Questions ==



= Las cartas de tarot se presentan en formato de imagen? =
No. Decidimos no colocar imagenes porque cada tarotista tiene su mazo preferido. En 
cambio los nombres son universales, de modo que cada uno podra interpretarlos a su 
modo.
